The problem this week is downloadable as a PDF from the UVA website. You may recognize this as the 8-Queens problem, with a slight variation.

If you're programming in C, C++, Pascal, or Java, and you'd like to have their website verify the correctness of your program, you can sign up for an account on their website and submit a response to this problem. If you use Java, make sure to name your class 'Main' with no package.

Be very aware of the ordering of the solutions you provide. We will be checking this :). You can feel free to omit the header output if you wish.

For an extra challenge, try and structure your program so that I could change one constant (or better yet, take a runtime input) and use the program to solve any of the N-queens problems.

When you get your program running with the sample input provided in the problem PDF, submit your program source code and the output when you run it with the below input to codechallenge@sudounlv.com:
